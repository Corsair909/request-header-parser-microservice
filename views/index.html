Request Header Parser Microservice

Microservice to parse request headers

Usage:

Runs on port 3000 by default or PORT env variable if set.

[URL]/api/whoami/

Responds with: 

          {
            "ipaddress": your public ip address,
            "language": your language,
            "software": your operating system
          }
